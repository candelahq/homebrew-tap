# Candela Homebrew Tap

Homebrew formulae and casks for [Candela](https://github.com/candelahq/candela) — LLM observability for your machine.

## Install

```bash
brew tap candelahq/tap
brew install --cask candela
```

## What's Available

| Formula/Cask | Type | Description |
|---|---|---|
| `candela` | Cask | Candela Desktop — native macOS app with system tray, traces, budgets |

## Upgrade

```bash
brew upgrade --cask candela
```

## Uninstall

```bash
brew uninstall --cask candela
brew untap candelahq/tap
```
